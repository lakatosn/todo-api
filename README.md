# To-Do List API

API simples de gerenciamento de tarefas (CRUD) desenvolvida com Flask e SQLite.

## Funcionalidades
- Criar tarefas
- Listar tarefas
- Atualizar tarefas
- Deletar tarefas

## Tecnologias
- Python
- Flask
- SQLite

## Como rodar
1. pip install -r requirements.txt
2. python init_db.py
3. python app.py
